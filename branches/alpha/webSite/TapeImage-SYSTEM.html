Burroughs B5500 Mark XIII SYSTEM Tape Image

The SYSTEM tape image contains object code for the Mark XIII release, including the Datacom MCP, Timesharing MCP, System Intrinsics, CANDE timesharing system, compilers, and utility programs. It also contains source files for a few utilities, and a set of source-level patch files for the release. A directory of the files on the image appears below. The download .zip file size is 2.6MiB, expanding to 12.4MiB.
