Main Links

    Describes the resources you will need to set up and run the emulator, along with instructions for downloading the necessary files and doing the setup.

    Describes how to configure system components and disk subsystems for the emulator.

    Describes how to operate the supervisory keyboard/printer within the browser-based emulator. Also gives a brief overview of MCP commands used with the SPO.
  • Using the Card Reader
    Describes how to operate the card reader device within the browser-based emulator.
  • Using the Line Printer
    Describes how to operate the line printer within the browser-based emulator.
  • Using the Card Punch
    Describes how to operate the card punch device within the browser-based emulator.
  • Using the Magnetic Tape Drive
    Describes how to operate magnetic tape drives within the browser-based emulator.
  • Using the Datacom Terminal
    Describes how to operate the data-communications terminal within the browser-based emulator.
  • Setting Up TSMCP and CANDE
    Describes how to set up and initiate the Timesharing MCP and CANDE user interface within the browser-based emulator.
